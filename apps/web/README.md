# Casino Frontend

React + Vite frontend for the Fare Casino platform.

## Quick Start

```bash
# Install dependencies (from root)
cd ../..
npm install

# Setup environment
cd apps/web
cp .env.example .env
# Edit .env with your API URLs

# Start development server
npm run dev
```

## Available at
- Frontend: http://localhost:3000
